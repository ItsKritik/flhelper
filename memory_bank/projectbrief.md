# Project Brief

Основная цель проекта: Создание генератора паттернов для FL-Studio с поддержкой различных гармонических структур и шкал.

## Project Deliverables

| ID | Название | Статус | Вес |
|----|----------|--------|-----|
| D1 | Фаза 0: HTML/CSS/JS прототип | completed | 10% |
| D2 | Фаза 1: Настройка TypeScript, структура папок, тесты | completed | 10% |
| D3 | Фаза 2: Модели данных и валидация | completed | 15% |
| D4 | Фаза 3: Движок генерации (гаммы, гармония) | completed | 20% |
| D5 | Фаза 4: UI компоненты (Canvas, панели) | completed | 20% |
| D6 | Фаза 5: MIDI экспорт и сохранение | completed | 10% |
| D7 | Фаза 6: Пресеты | completed | 10% |
| D8 | Фаза 7: Финальная проверка | in_progress | 5% |

Процент выполнения: 95%
