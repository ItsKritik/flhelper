# Active Context

Текущий фокус работы: исправление отображения нотных строк и улучшение коротких паттернов.

Активные решения и приоритеты:
- Показывать в piano roll только реально используемые нотные строки без пустых рядов.
- Исправить соответствие `pitch` и визуальной строки, чтобы активные ноты попадали в правильные позиции.
- Улучшить генерацию коротких паттернов, особенно на длине `4`, чтобы шаги не оставались пустыми.
- Согласовать мелодическое движение с базовой логикой мажорных и минорных схем.

Предыдущие достижения:
- Реализованы генерация паттернов, базовый UI, экспорт и пресеты.
- Обновлен интерфейс главной страницы без изменения архитектуры приложения.
- Генератор паттернов переведен с хаотичного заполнения на более структурированную мелодическую логику.
- Piano roll переведен на рендер только используемых рядов по фактическим `pitch/octave` из паттерна.
