# Active Context

Текущий фокус работы: улучшение визуального дизайна основного экрана генератора паттернов.

Активные решения и приоритеты:
- Сохранить существующий сценарий генерации без изменения бизнес-логики.
- Улучшить визуальную иерархию главного экрана: hero-блок, карточки параметров, аналитическая панель.
- Сделать piano roll визуально чище и удобнее для чтения ритма.
- Поддерживать оба варианта темы через `prefers-color-scheme` в `style.css`.

Предыдущие достижения:
- Реализованы генерация паттернов, базовый UI, экспорт и пресеты.
- Обновлен интерфейс главной страницы без изменения архитектуры приложения.
