# Product Context

Проблемы пользователей: Музыканты нуждаются в инструменте для генерации паттернов с поддержкой различных гармонических структур.

Зачем нужен проект: Автоматизация создания музыкальных паттернов для FL-Studio.

Основные функции:
- Генерация паттернов на основе гармонических структур
- Поддержка различных музыкальных шкал
- Экспорт в MIDI формат
- Управление предустановками
