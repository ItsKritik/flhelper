## Changelog

- Создана структура Memory Bank
- Создан docs/README.md с описанием архитектуры
- Выполнена Фаза 1: инициализация npm/bun, установка зависимостей (TypeScript, Jest, ts-jest, midi-file, http-server)
- Создана структура папок: src/core, src/ui, src/utils, src/models, src/presets
- 12 unit-тестов проходят
- Сборка TypeScript работает без ошибок
- Обновлена главная страница: добавлены hero-секция, карточки параметров и улучшенная аналитическая панель.
- Переработаны стили piano roll и сетки с учетом темной и светлой темы.
- После изменений выполнены `biome` с автоисправлением и `bun run build`.
- Убраны градиенты у основных кнопок в пользу однотонной акцентной заливки.
- Генерация нот в `src/core/generator.ts` переведена на фразовую логику с опорой на сильные доли, соседнее движение и финальное разрешение.
- В `src/ui/gridRenderer.ts` убраны лишние пустые нотные строки: теперь рендерятся только реально используемые `pitch/octave`.
- Исправлен маппинг названий нот и `pitch`, из-за которого активные ноты могли визуально попадать не в те строки.
- Для длины паттерна `4` добавлен отдельный короткий мелодический шаблон без пропусков шагов.
