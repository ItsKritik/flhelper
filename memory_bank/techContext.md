# Tech Context

## Стек технологий

- TypeScript
- Jest (тестирование)
- MIDI file handling (midi-file)
- Node.js
- Vite для dev-режима фронтенда
- Biome для линтинга и автоисправления

## Окружение

- Пакетный менеджер: bun
- Сборка: tsc (TypeScript compiler)
- Dev-сервер: Vite
- Статический сервер: http-server

## Ограничения

- CommonJS модули
- Требуется Node.js для сборки
- Markdown-файлы не должны проверяться через Biome
