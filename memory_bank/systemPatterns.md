# System Patterns

## Архитектура

Основные модули:
- `core/` - ядро генерации паттернов
- `ui/` - пользовательский интерфейс
- `presets/` - управление предустановками
- `utils/` - утилиты для работы с файлами и MIDI

Текущий UI-паттерн:
- Главная страница построена как одностраничный экран с hero-секцией, панелью управления, областью piano roll и боковой аналитикой.
- `src/index.ts` отвечает за инициализацию интерфейса, синхронизацию контролов и рендер информационной панели.
- `src/ui/gridRenderer.ts` рендерит piano roll напрямую через DOM и inline-стили, используя CSS-переменные из `style.css`.

## Связи подсистем

- generator.ts зависит от harmony.ts и scale.ts
- ui/依赖于 core/ для генерации паттернов
- midiExporter.ts использует models/pattern.ts
- Визуальная тема задается глобальными CSS-переменными и автоматически переключается через `prefers-color-scheme`.
